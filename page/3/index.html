и®Ўз»„д№ йўҳзә й”ҷжҖ»з»“

Posted on 2019-09-22 | In и®Ўз®—жңәз»„жҲҗеҺҹзҗҶ

дҫӢ2.6 жҹҗжңәеӯ—й•ҝ32дҪҚпјҢжө®зӮ№иЎЁзӨәж—¶пјҢжҢҮж•°йғЁеҲҶпјҲеҚійҳ¶з ҒпјүеҚ 8дҪҚпјҲеҗ«дёҖдҪҚз¬ҰеҸ·дҪҚпјүпјҢе°ҫж•°йғЁеҲҶеҚ 24дҪҚпјҲеҗ«дёҖдҪҚз¬ҰеҸ·дҪҚпјүпјҢй—®:

  1. еёҰз¬ҰеҸ·е®ҡзӮ№е°Ҹж•°зҡ„жңҖеӨ§иЎЁзӨәиҢғеӣҙжҳҜеӨҡе°‘пјҹ
  2. еёҰз¬ҰеҸ·е®ҡзӮ№ж•ҙж•°зҡ„жңҖеӨ§иЎЁзӨәиҢғеӣҙжҳҜеӨҡе°‘пјҹ
  3. жө®зӮ№иЎЁзӨәж—¶пјҢжңҖеӨ§жӯЈж•°жҳҜеӨҡе°‘пјҹ
  4. жө®зӮ№иЎЁзӨәж—¶пјҢжңҖеӨ§иҙҹж•°жҳҜеӨҡе°‘пјҹ
  5. жө®зӮ№иЎЁзӨәж—¶пјҢжңҖе°Ҹзҡ„и§„ж јеҢ–жӯЈж•°жҳҜеӨҡе°‘пјҹ

еҲҶжһҗпјҡ

йҮҮз”ЁиЎҘз ҒиЎЁзӨәж—¶пјҢеҸҜиЎЁзӨәзҡ„ж•°жҚ®иҢғеӣҙжңҖеӨ§гҖӮеӣ жӯӨжң¬йўҳеҸҜз”ЁиЎҘз ҒжқҘжұӮи§ЈгҖӮ

еёҰз¬ҰеҸ·зҡ„е®ҡзӮ№е°Ҹж•°йҖҡеёёе°ұжҳҜжө®зӮ№ж•°гҖӮ
жө®зӮ№ж•°жҚ®иЎЁзӨәйҳ¶з ҒеҚ 8дҪҚпјҢе°ҫж•°еҚ 24дҪҚпјҢеҗ„еҢ…еҗ«дёҖдҪҚз¬ҰеҸ·дҪҚгҖӮжӯӨж—¶йҳ¶з Ғзҡ„иЎЁзӨәиҢғеӣҙдёәпјҡ-128 пҪһ 127пјӣе°ҫж•°зҡ„иЎЁзӨәиҢғеӣҙдёәпјҡ-1 пҪһ 1-$2 ^{вҖ“ 23}$гҖӮ

еёҰз¬ҰеҸ·зҡ„е®ҡзӮ№ж•ҙж•°е…¶з¬ҰеҸ·дҪҚ1дҪҚпјҢе…¶дҪҷдҪҚеқҮз”ЁжқҘиЎЁзӨәж•°жҚ®гҖӮжӯӨж—¶зҡ„иЎЁзӨәиҢғеӣҙдёәпјҡ-$2^{31}$ пҪһ $2^{31}$-1гҖӮ

жө®зӮ№иЎЁзӨәж—¶пјҢжңҖеӨ§жӯЈж•°дёәйҳ¶з ҒжңҖеӨ§пјҢе°ҫж•°жңҖеӨ§зҡ„жӯЈж•°гҖӮжңҖеӨ§иҙҹж•°дёәйҳ¶з ҒжңҖе°ҸпјҢе°ҫж•°жңҖе°Ҹзҡ„иҙҹж•°гҖӮ
йҮҮз”Ёи§„ж јеҢ–ж•°жҚ®иЎЁзӨәж—¶иҰҒжіЁж„Ҹе°ҫж•°еҝ…йЎ»дҝқиҜҒеӨ§дәҺзӯүдәҺ0.5пјҢе°ҸдәҺ1гҖӮеӣ жӯӨпјҢжө®зӮ№иЎЁзӨәж—¶пјҢжңҖе°Ҹзҡ„и§„ж јеҢ–жӯЈж•°йҳ¶з ҒжңҖе°ҸпјҢе°ҫж•°жңҖе°Ҹдёә0.5гҖӮ

и§Јпјҡ

йҮҮз”ЁиЎҘз ҒиҝӣиЎҢж•°жҚ®иЎЁзӨәзҡ„иҢғеӣҙжңҖеӨ§гҖӮж•ҙж•°иЎЁзӨәж—¶з¬ҰеҸ·дҪҚ1дҪҚпјҢж•°жҚ®дҪҚ31дҪҚгҖӮжө®зӮ№иЎЁзӨәж—¶пјҢйҳ¶з ҒеҚ 8дҪҚпјҢе°ҫж•°еҚ 24дҪҚпјҢеҗ„еҢ…еҗ«дёҖдҪҚз¬ҰеҸ·дҪҚгҖӮеӣ жӯӨпјҡ

  1. еёҰз¬ҰеҸ·е®ҡзӮ№е°Ҹж•°зҡ„жңҖеӨ§иЎЁзӨәиҢғеӣҙдёәпјҡ-1Г—$2^{127}$ пҪһ пјҲ1-$2^{-23}$пјүГ—$2^{127}$гҖӮ
  2. еёҰз¬ҰеҸ·е®ҡзӮ№ж•ҙж•°зҡ„жңҖеӨ§иЎЁзӨәиҢғеӣҙдёәпјҡ-$2^{31}$ пҪһ $2^{31}$-1гҖӮ
  3. жө®зӮ№иЎЁзӨәж—¶пјҢжңҖеӨ§жӯЈж•°дёәпјҡпјҲ1-$2^{-23}$пјүГ—$2^{127}$гҖӮ
  4. жө®зӮ№иЎЁзӨәж—¶пјҢжңҖеӨ§иҙҹж•°дёәпјҡ-$2^{-23}$Г—$2^{-128}$гҖӮ
  5. жө®зӮ№иЎЁзӨәж—¶пјҢжңҖе°Ҹзҡ„и§„ж јеҢ–жӯЈж•°дёәпјҡ0.5Г—$2^{-128}$гҖӮ

classзұ»

java.lang.Classзұ»е°ҒиЈ…дёҖдёӘеҜ№иұЎе’ҢжҺҘеҸЈиҝҗиЎҢж—¶зҡ„зҠ¶жҖҒгҖӮ

еҪ“еҠ иҪҪзұ»ж—¶Classзұ»еһӢзҡ„еҜ№иұЎе°ҶиҮӘеҠЁеҲӣе»әпјҢйҖҡиҝҮClassеҜ№иұЎе°ұеҸҜд»Ҙи®ҝй—®JVMдёӯзұ»зҡ„жүҖжңүдҝЎжҒҜгҖӮ

Classзұ»жІЎжңүе…¬е…ұжһ„йҖ ж–№жі•пјҢе…¶еҜ№иұЎжҳҜJVMеңЁеҠ иҪҪзұ»ж—¶йҖҡиҝҮзұ»еҠ иҪҪеҷЁдёӯзҡ„defineClassпјҲпјүж–№жі•иҮӘеҠЁжһ„йҖ зҡ„пјҢеӣ жӯӨдёҚиғҪжҳҫејҸең°жһ„йҖ дёҖдёӘClassеҜ№иұЎгҖӮ

еҸҜд»ҘиҺ·еҸ–жүҖжңүжһ„йҖ ж–№жі•пјҢж–№жі•пјҢеҸҳйҮҸпјҢеҶ…йғЁзұ»пјҢеӨ–йғЁзұ»пјҢеҜ№еә”зұ»зҡ„дҝ®йҘ°з¬ҰпјҢе№¶дё”еҲӨж–ӯзұ»еһӢгҖӮ

иҺ·еҸ–ClassеҜ№иұЎжңүдёүз§Қж–№жі•:

  1. Class.forName(вҖңжүҖйңҖзұ»зҡ„е…Ёйҷҗе®ҡзұ»еҗҚвҖқ)
  2. и°ғз”ЁжҹҗдёӘзұ»зҡ„classж–№жі• Float.class
  3. и°ғз”ЁжҹҗдёӘеҜ№иұЎзҡ„getClassпјҲпјүж–№жі• a.getClass()

зұ»еҠ иҪҪжӯҘйӘӨ

еҪ“зЁӢеәҸдё»еҠЁдҪҝз”ЁжҹҗдёӘзұ»ж—¶пјҢеҰӮжһңиҜҘзұ»иҝҳжңӘиў«еҠ иҪҪеҲ°еҶ…еӯҳдёӯпјҢзі»з»ҹе°ҶйҖҡиҝҮеҠ иҪҪпјҢиҝһжҺҘе’ҢеҲқе§ӢеҢ–дёүдёӘжӯҘйӘӨеҜ№зұ»иҝӣиЎҢеҲқе§ӢеҢ–гҖӮ

зұ»еҠ иҪҪеҷЁ

зұ»еҠ иҪҪеҷЁиҙҹиҙЈе°ҶзЈҒзӣҳжҲ–иҖ…зҪ‘з»ңдёҠзҡ„.classж–Үд»¶еҠ иҪҪеҲ°еҶ…еӯҳдёӯпјҢе№¶дёәд№Ӣз”ҹжҲҗеҜ№еә”зҡ„java.lang.ClassеҜ№иұЎ

зұ»еҠ иҪҪеҷЁиҙҹиҙЈеҠ иҪҪжүҖжңүзҡ„зұ»,JVMеҗҜеҠЁж—¶пјҢдјҡеҪўжҲҗз”ұдёүдёӘзұ»еҠ иҪҪеҷЁз»„жҲҗзҡ„еҲқе§Ӣзұ»еҠ иҪҪеҷЁеұӮж¬Ўз»“жһ„пјҡ

  • Bootstrap ClassLoaderпјҡж №зұ»еҠ иҪҪеҷЁпјҢиҙҹиҙЈеҠ иҪҪJavaзҡ„ж ёеҝғзұ»еә“
  • Extension ClassLoaderпјҡжү©еұ•зұ»еҠ иҪҪеҷЁпјҢиҙҹиҙЈеҠ иҪҪJREзҡ„жү©еұ•зӣ®еҪ•дёӯзҡ„jarеҢ…
  • System ClassLoaderпјҡзі»з»ҹзұ»еҠ иҪҪеҷЁпјҢиҙҹиҙЈеңЁJVMеҗҜеҠЁж—¶еҠ иҪҪжқҘиҮӘJavaе‘Ҫд»Өзҡ„-classpathйҖүйЎ№пјҢjava.class.pathзі»з»ҹеұһжҖ§пјҢжҲ–CLASSPATHзҺҜеўғеҸҳйҮҸжүҖжҢҮе®ҡзҡ„jarеҢ…е’Ңзұ»и·Ҝеҫ„
  • з”ЁжҲ·д№ҹеҸҜд»ҘиҮӘе®ҡд№үеҠ иҪҪеҷЁ

tipsпјҡзұ»еҠ иҪҪжңәеҲ¶пјҡе…ЁзӣҳиҙҹиҙЈпјҢзҲ¶зұ»е§”жүҳпјҢзј“еӯҳжңәеҲ¶

ClassLoaderзұ»

java.lang.ClassLoaderжҳҜдёҖдёӘжҠҪиұЎзұ»пјҢйҖҡиҝҮз»§жүҝClassLoaderзұ»жқҘе®һзҺ°иҮӘе®ҡд№үзҡ„з”ЁжҲ·зұ»еҠ иҪҪеҷЁпјҢйҮҚеҶҷж–№жі•
е®һзҺ°иҮӘе®ҡд№үзҡ„зұ»еҠ иҪҪеҷЁеҸҜд»ҘйҖҡиҝҮйҮҚеҶҷClassLoaderзұ»зҡ„loadClass()жҲ–findClassпјҲпјүж–№жі•жқҘе®һзҺ°гҖӮжҺЁиҚҗйҮҚеҶҷfindClass()йҒҝе…ҚиҰҶзӣ–й»ҳи®Өзұ»еҠ иҪҪеҷЁзҡ„зҲ¶зұ»е§”жүҳпјҢзј“еҶІжңәеҲ¶дёӨз§Қзӯ–з•Ҙ

еҸҚе°„

javaдёӯжңүи®ёеӨҡеҜ№иұЎеңЁиҝҗиЎҢж—¶йғҪдјҡеҮәзҺ°дёӨз§Қзұ»еһӢпјҡзј–иҜ‘ж—¶зұ»еһӢе’ҢиҝҗиЎҢж—¶зұ»еһӢпјҢ

жҹҗдәӣжғ…еҶөдёӢзЁӢеәҸдјҡеңЁиҝҗиЎҢж—¶жҺҘж”¶еҲ°еӨ–йғЁдј е…Ҙзҡ„дёҖдёӘеҜ№иұЎпјҢиҜҘеҜ№иұЎзҡ„зј–иҜ‘ж—¶зұ»еһӢе’ҢиҝҗиЎҢж—¶зұ»еһӢдёҚеҗҢпјҢ
дҪҶзЁӢеәҸйңҖиҰҒи°ғз”ЁиҜҘеҜ№иұЎиҝҗиЎҢж—¶зұ»еһӢзҡ„ж–№жі•пјҢеңЁж— жі•йў„зҹҘиҜҘеҜ№иұЎеұһдәҺе“Әдәӣзұ»зҡ„жғ…еҶөдёӢпјҢеҝ…йЎ»йҖҡиҝҮеҸҚе°„жқҘеҸ‘зҺ°иҜҘеҜ№иұЎе’Ңзұ»зҡ„зңҹе®һдҝЎжҒҜгҖӮ

еҸҚе°„жңәеҲ¶е…Ғи®ёзЁӢеәҸеңЁиҝҗиЎҢж—¶еҖҹеҠ©Reflection APIеҸ–еҫ—д»»дҪ•зұ»зҡ„еҶ…йғЁдҝЎжҒҜпјҢе№¶иғҪзӣҙжҺҘж“ҚдҪңеҜ№иұЎзҡ„еҶ…йғЁеұһжҖ§е’Ңж–№жі•гҖӮеҸҚе°„иў«и§ҶдёәеҠЁжҖҒиҜӯиЁҖзҡ„е…ій”®гҖӮ

javaеҸҚе°„жңәеҲ¶дё»иҰҒжҸҗдҫӣдәҶд»ҘдёӢеҠҹиғҪпјҡ

  • еңЁиҝҗиЎҢж—¶еҲӨж–ӯд»»ж„ҸдёҖдёӘеҜ№иұЎжүҖеұһзҡ„зұ»
  • еңЁиҝҗиЎҢж—¶жһ„йҖ д»»ж„ҸдёҖдёӘзұ»зҡ„еҜ№иұЎ
  • еңЁиҝҗиЎҢж—¶иҺ·еҸ–д»»ж„ҸдёҖдёӘзұ»жүҖе…·жңүзҡ„жҲҗе‘ҳеҸҳйҮҸе’Ңж–№жі•
  • еңЁиҝҗиЎҢж—¶и°ғз”Ёд»»ж„ҸдёҖдёӘеҜ№иұЎзҡ„ж–№жі•
  • з”ҹжҲҗеҠЁжҖҒд»ЈзҗҶ

Reflection APIжҸҗдҫӣдәҶConstructorпјҢFieldе’ҢMethodзұ»пјҢиҝҷдёүдёӘзұ»е®ҡд№үеңЁjava.lang.reflectеҢ…дёӯпјҢеҲҶеҲ«з”ЁдәҺжҸҸиҝ°зұ»зҡ„жһ„йҖ ж–№жі•пјҢеұһжҖ§е’Ңж–№жі•

java.lang.reflectеҢ…дёӢзҡ„ExecutableжҠҪиұЎзұ»пјҢиҜҘеҜ№иұЎд»ЈиЎЁеҸҜжү§иЎҢзҡ„зұ»жҲҗе‘ҳгҖӮ
ExecutableжҠҪиұЎзұ»жҙҫз”ҹдәҶConstructorе’ҢMethodдёӨдёӘеӯҗзұ»гҖӮ

ExecutableжҠҪиұЎзұ»жҸҗдҫӣдәҶеӨ§йҮҸж–№жі•жқҘиҺ·еҸ–еҸӮж•°пјҢдҝ®йҘ°з¬ҰжҲ–жіЁи§ЈзӯүдҝЎжҒҜ

Executableж–№жі•еҲ—иЎЁ

ж–№жі• еҠҹиғҪжҸҸиҝ°
Parameter[] getParameters() иҺ·еҸ–жүҖжңүеҪўеҸӮпјҢиҝ”еӣһдёҖдёӘParameter[]ж•°з»„
int getParameterCount() иҺ·еҸ–еҪўеҸӮдёӘж•°
abstract int getModifiers иҺ·еҸ–дҝ®йҘ°з¬ҰпјҢиҝ”еӣһзҡ„ж•ҙж•°жҳҜдҝ®йҘ°з¬ҰжүҖеҜ№еә”зҡ„еёёйҮҸ
boolean isVarArgsпјҲпјү еҲӨж–ӯжҳҜеҗҰеҢ…еҗ«ж•°йҮҸеҸҜеҸҳзҡ„еҪўеҸӮ

Constructorзұ»з”ЁдәҺиЎЁзӨәзұ»зҡ„жһ„йҖ ж–№жі•пјҢйҖҡиҝҮи°ғз”ЁClassеҜ№иұЎзҡ„getConstructor()ж–№жі•еҸҜд»ҘиҺ·еҸ–еҪ“еүҚзұ»зҡ„жһ„йҖ ж–№жі•зҡ„йӣҶеҗҲ

Constructorзұ»ж–№жі•еҲ—иЎЁ

ж–№жі• еҠҹиғҪжҸҸиҝ°
String getName() иҝ”еӣһжһ„йҖ ж–№жі•зҡ„еҗҚз§°
Class[] getParameterTypes() иҝ”еӣһеҪ“еүҚжһ„йҖ ж–№жі•зҡ„еҸӮж•°зұ»еһӢ
int getModifiers() иҝ”еӣһдҝ®йҘ°з¬Ұзҡ„ж•ҙеһӢж ҮиҜҶпјҢиҝ”еӣһзҡ„ж•ҙж•°жҳҜдҝ®йҘ°з¬ҰpublicпјҢprotectedпјҢprivateпјҢfinalпјҢstaticпјҢabstractзӯүе…ій”®еӯ—жүҖеҜ№еә”зҡ„еёёйҮҸпјҢйңҖиҰҒдҪҝз”ЁModifierе·Ҙе…·зұ»зҡ„ж–№жі•и§Јз ҒеҗҺжүҚиғҪиҺ·еҫ—зңҹе®һзҡ„дҝ®йҘ°з¬Ұ

Methodзұ»з”ЁдәҺе°ҒиЈ…ж–№жі•зҡ„дҝЎжҒҜпјҢи°ғз”ЁClassеҜ№иұЎзҡ„getMethodsпјҲпјүж–№жі•жҲ–getMethodпјҲпјүеҸҜд»ҘиҺ·еҸ–еҪ“еүҚзұ»зҡ„жүҖжңүж–№жі•жҲ–жҢҮе®ҡж–№жі•
Methodзұ»еёёз”Ёж–№жі•

ж–№жі• еҠҹиғҪжҸҸиҝ°
String getNameпјҲпјү иҝ”еӣһж–№жі•зҡ„еҗҚз§°
Class[] getParameterTypesпјҲпјү иҝ”еӣһеҪ“еүҚж–№жі•зҡ„еҸӮж•°зұ»еһӢ
int getModifiersпјҲпјү иҝ”еӣһдҝ®йҘ°з¬Ұзҡ„ж•ҙеһӢж ҮиҜҶ
Class getReturnTypeпјҲпјү иҝ”еӣһеҪ“еүҚж–№жі•зҡ„иҝ”еӣһзұ»еһӢ

Fieldзұ»з”ЁдәҺе°ҒиЈ…еұһжҖ§зҡ„дҝЎжҒҜпјҢи°ғз”ЁClassеҜ№иұЎзҡ„getFieldsпјҲпјүжҲ–getFieldпјҲпјүж–№жі•еҸҜд»ҘиҺ·еҫ—еҪ“еүҚзұ»зҡ„жүҖжңүеұһжҖ§жҲ–жҢҮе®ҡеұһжҖ§гҖӮ

Fieldзұ»еёёз”Ёж–№жі•

ж–№жі• еҠҹиғҪжҸҸиҝ°
String getNameпјҲпјү иҝ”еӣһеұһжҖ§зҡ„еҗҚз§°
int getModifiersпјҲпјү иҝ”еӣһдҝ®йҘ°з¬Ұзҡ„ж•ҙеһӢж ҮиҜҶ
getXxxпјҲObject objпјү иҺ·еҸ–еұһжҖ§зҡ„еҖјпјҢжӯӨеӨ„зҡ„XxxеҜ№еә”javaдёӯеҹәжң¬зұ»еһӢпјӣеҰӮжһңеұһжҖ§жҳҜеј•з”Ёзұ»еһӢпјҢеҲҷзӣҙжҺҘдҪҝз”ЁgetпјҲObject objпјү

Parameterзұ»пјҢжҜҸдёӘParameterеҜ№иұЎд»ЈиЎЁж–№жі•зҡ„дёҖдёӘеҸӮж•°гҖӮParameterзұ»дёӯжҸҗдҫӣдәҶи®ёеӨҡж–№жі•жқҘиҺ·еҸ–еҸӮж•°дҝЎжҒҜгҖӮ

Parameterзұ»еёёз”Ёж–№жі•

ж–№жі• еҠҹиғҪжҸҸиҝ°
int getModifiersпјҲпјү иҺ·еҸ–еҸӮж•°зҡ„дҝ®йҘ°з¬Ұ
